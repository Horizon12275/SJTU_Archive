## 本份README文档说明了如何编译、运⾏测试、获取实验数据结果。

运行环境：Ubuntu 22.04虚拟环境；虚拟机可用磁盘大小：32GB；虚拟机内存：8GB；虚拟机处理器内核总数：6个

# 正确性测试方法

在根目录下运行如下指令

1. make clean

2. make all

3. make grade

得到正确性测试全部通过，为100分

# 第三部分测试方法

在./test目录下运行如下指令

1. make clean

2. make

3. ./main

4. 等一会就能跑出所有结果了，如要修改运行时间，在./test/main.cpp里的宏RUN_TIME中进行修改，单位为秒，默认值为60
